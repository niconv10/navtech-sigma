# CLAUDE.md — NavTech SIGMA

## What is SIGMA?
SIGMA is a student productivity platform that integrates with Canvas LMS to automatically surface assignments, deadlines, and grades. Built by NavTech (Nicolas Navarro). Primary target: community college students in Florida.

## Tech Stack
- Framework: Next.js 14+ (App Router, TypeScript)
- Database: Supabase (PostgreSQL + Auth + Realtime)
- Styling: Tailwind CSS
- Payments: Stripe (Phase 3)
- Hosting: Vercel
- AI: Claude API Sonnet (Pro features only)

## Key Rules
- SIGMA parses a syllabus (singular), NOT syllabi
- NavTech is the parent company
- Community colleges are the primary target market
- Quality standard: Apple-level UX
- Mobile-first PWA

## Current Phase
Phase 0: Foundation & Structure (no app code yet)

## Market
- Florida College System: 28 state colleges, 813,000+ students
- All use Canvas LMS
- Starting with: FAU (beta), then Broward College, Miami Dade College, Palm Beach State

## Business Model
- Free tier: 2 courses max
- Pro tier: $4.99/mo — unlimited courses, AI study assistant, GPA predictor
